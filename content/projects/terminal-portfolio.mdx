## Overview

A statically-exported personal site with a dark, terminal-inspired aesthetic.
The blog is powered by MDX files grouped into topics, and projects (like this
one) are authored the same way.

## Highlights

- **Zero-backend**: fully static, deployable to GitHub Pages.
- **Interactive resume** that highlights each role as you scroll.
- **Topic-grouped blog** with Shiki-highlighted code blocks.
- **Cursor-tracking hover effects** on every card.

## How it works

Content lives in plain Markdown/MDX under `content/`. A small filesystem loader
reads frontmatter at build time and `generateStaticParams` pre-renders every
page — no database, no server.
